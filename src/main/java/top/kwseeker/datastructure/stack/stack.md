## 栈 Stack

知道元素后进先出就够了吧，使用代码实现也比较简单。
使用数组或链表都可以实现。

#### 栈的实际应用
+ 撤销操作  
+ 程序调用的系统栈
+ 括号匹配
    如判断一段代码是否正确闭合,如下代码：使用括号为"([]){{}()}"是闭合的
    ```
    public static void main(String[] args) {
        {
            int a=0;
            a++;
        }
        System.out.println("Hello world");
    }
    ```

有个关于栈的经典面试题，好像说的是  
如何实现栈？然后如何记录栈的最小值？  
通过辅助数组记录最小值的索引。  
这个问题还有另一种说法就是：实现一个栈，要求Push(入栈)，Pop(出栈)，Min(返回最小值的操作)的时间复杂度为O(1)。
